# Future Purpose Sketches

These purpose sketches are not implemented yet. Use them to keep Lookbook Image from becoming too narrow.

## Cast Portrait

Possible key: `cast.portrait`

Purpose: create a single portrait that demonstrates a cast member's presence, styling, and emotional register.

Likely context: cast profile, role in story, age/presentation notes, screenplay mentions, existing selected images, project visual language.

## Cast Character Sheet

Possible key: `cast.character-sheet`

Purpose: create a sheet of expressions, wardrobe, pose language, and surface details for a cast member.

Likely context: cast profile, wardrobe notes, scene contexts, Lookbook, selected portraits.

## Scene Narration Audio

Possible key: `scene.narration-audio`

Purpose: create narration, temporary voiceover, or guide audio for a scene.

Likely context: scene narrative, dialogue, tone, target language, cast voice references when approved by the user.
