# Cinematography Analysis Guidelines

Write for working cinematographers. Be concrete, visual, and useful.

## Use The Folder Name

Use the folder name as an important creative and research hint. It is often a movie title, director, cinematographer, photographer, painter, movement, location, period, or production phrase.

If the folder name points to a known reference, use reliable context to improve the analysis:

- known collaborators;
- era and format;
- genre grammar;
- director or cinematographer signatures;
- production context;
- recurring visual strategies;
- relevant movements in photography, painting, or design.

Let that context guide what to look for, but keep the persisted analysis grounded in the actual images in the folder.

Example: if the folder is named `Blade Runner 2049`, it is useful to bring knowledge of Roger Deakins, haze, controlled color fields, large-scale lighting, and monumental negative space. Still, each persisted claim should be supported by the folder images.

## Evidence Rules

- Inspect every supported image file in the folder.
- Study the folder as a system, not isolated attractive frames.
- Cite image filenames only when they visibly demonstrate the exact point.
- Prefer 2-4 supporting images per observation when available.
- One strong image is better than four weak ones.
- Separate visible observation from contextual knowledge or plausible affinity.
- Do not present uncertain credits, influence, or production history as fact.

## Section Guidance

- `thesis`: summarize the visual argument of the folder.
- `palette`: explain color behavior, not just color names.
- `toneMood`: discuss exposure, contrast, saturation, day/night behavior, and emotional temperature.
- `composition`: identify repeatable framing, blocking, scale, geometry, and negative-space strategies.
- `lighting`: describe source logic, direction, falloff, contrast, fill, motivated light, and shadow behavior.
- `texture`: describe grain, surface, weather, lens/filter feel, tactility, and production texture.
- `inspiredBy`: name visual affinities cautiously. Use "potentially echoes", "shares a strategy with", or "sits near" unless the user supplied confirmed source information.

Avoid generic film-school labels unless you explain what the strategy does in this specific folder.
